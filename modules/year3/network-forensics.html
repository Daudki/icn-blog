Overview

Forensics depends on careful evidence handling and a clear timeline. Good investigations connect technical artifacts to understandable events.

Projects

Build simple tools to explore logs and sequence evidence.

Timeline Builder

Beginner

Organize a list of events into a readable timeline.

timeline.py
events = [('10:00', 'Login'), ('10:05', 'Logout')]
for t, action in events:
    print(t, action)

Log Parser

Beginner

Extract suspicious entries from a small log list.

log_parser.py
logs = ['INFO login', 'WARN failed login', 'INFO logout']
for entry in logs:
    if 'WARN' in entry:
        print(entry)
