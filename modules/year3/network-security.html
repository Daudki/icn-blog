Overview

Network security combines technical controls and thoughtful design to reduce risk. Even small habits like stronger passwords and log review matter a lot.

Projects

Try a few hands-on exercises that model common defense tasks.

Password Strength Checker

Beginner

Evaluate a password against simple strength rules.

password_check.py
def check_password_strength(password):
    return len(password) >= 12 and any(c.isdigit() for c in password)
print(check_password_strength('SecurePass2026'))
