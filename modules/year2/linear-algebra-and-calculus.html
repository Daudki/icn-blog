Overview

Mathematics underpins many computing tools. This module shows how vectors, matrices, and derivatives can be worked with in a simple, practical way.

Projects

Explore basic mathematical concepts through small numerical examples.

Vector Operations

Beginner

Use NumPy to add and scale vectors.

vector_ops.py
import numpy as np
v1 = np.array([1, 2, 3])
v2 = np.array([4, 5, 6])
print(v1 + v2)
print(v1 * 2)

Slope Explorer

Beginner

Approximate the slope of a simple function over a small interval.

slope_explorer.py
x1, x2 = 2, 4
f1, f2 = 4, 16
slope = (f2 - f1) / (x2 - x1)
print('Slope:', slope)
