SNMP Network Monitor

Intermediate

Query a device over SNMP and print its system name or uptime.

snmp_monitor.py
from pysnmp.hlapi import getCmd, SnmpEngine, CommunityData, UdpTransportTarget, ContextData, ObjectType, ObjectIdentity

result = getCmd(SnmpEngine(), CommunityData('public'), UdpTransportTarget(('192.168.1.1', 161)), ContextData(), ObjectType(ObjectIdentity('1.3.6.1.2.1.1.5.0')))
print(next(result))

Status Dashboard
