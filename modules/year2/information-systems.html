Overview

Information systems connect people, processes, and technology. This module shows that effective systems rely on clear data flows and well-designed tools.

Projects

Build small systems that capture and report data for daily operations.

Department Records Tracker

Beginner

Store department data in a dictionary and print summary reports.

records_tracker.py
records = {
    'IT': 12,
    'Admin': 8,
    'Library': 4
}
for dept, count in records.items():
    print(f'{dept}: {count}')

Summary Analyzer

Beginner

Summarize data into a few useful metrics for a simple management report.

summary_analyzer.py
values = [12, 8, 4, 15]
print('Total:', sum(values))
print('Average:', sum(values) / len(values))
