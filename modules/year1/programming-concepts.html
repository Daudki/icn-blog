Project: Automated File Organizer

Beginner 30 minutes

Build a Python script that automatically organizes files in a folder by their type...

What You'll Learn

  • Work with os and pathlib modules
  • Understand file system operations
  • Practice dictionary mapping
  • Handle file extensions

Step 1: Import Required Modules

import os
import shutil
from pathlib import Path

Line-by-Line Explanation

  1. import os — The os module provides functions to interact with the operating system. You'll use it for file operations like checking if files exist, creating directories, and getting file information.
  2. import shutil — Short for "shell utilities," this module offers high-level file operations. The key function you'll use is shutil.move() to relocate files between folders.
  3. from pathlib import Path — pathlib is the modern, object-oriented way to handle file paths. Instead of string manipulation ('/home/user/docs'), you work with Path objects that have useful methods like .suffix, .name, and .iterdir().

Step 2: Define the File Extension Map

FOLDER_MAP = {
    '.jpg': 'Images',
    '.png': 'Images',
    '.gif': 'Images',
    '.pdf': 'Documents',
    '.docx': 'Documents',
    '.mp3': 'Music',
    '.mp4': 'Videos',
    '.py': 'Code',
    '.zip': 'Archives'
}

Line-by-Line Explanation

  1. FOLDER_MAP = { ... } — This is a Python dictionary. Each key is a file extension (like '.pdf'), and each value is the destination folder name.
  2. Notice how multiple extensions ('.jpg', '.png', '.gif') all map to the same folder 'Images'. This groups related file types together.
  3. Why a dictionary? Dictionaries provide O(1) lookup time — checking where a .pdf file should go is instant, no matter how many extensions you add.
