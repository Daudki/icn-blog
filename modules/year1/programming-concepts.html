Automated File Organizer

Beginner

Build a script that sorts files into folders based on their extensions and learns how Python handles paths, dictionaries, and loops.

file_organizer.py 
import shutil
from pathlib import Path

folder_map = {
    '.jpg': 'Images',
    '.png': 'Images',
    '.pdf': 'Documents',
    '.py': 'Code'
}

for file in Path('.').iterdir():
    if file.is_file() and file.suffix in folder_map:
        target = Path(folder_map[file.suffix])
        target.mkdir(exist_ok=True)
        shutil.move(str(file), str(target / file.name))

Why it matters

  1. Path objects make file handling more readable than raw strings.
  2. Dictionary mappings turn repetitive rules into a simple structure.
  3. Loops and conditionals make the script useful in real folders right away.

System Monitor

Intermediate

Track basic system details like CPU load, memory usage, and open processes using Python’s built-in tools.

system_monitor.py 
import os
import time

while True:
    print("CPU:", os.cpu_count())
    print("Memory:", os.sys.getsizeof(os))
    time.sleep(1)

Why it matters

  1. It introduces the habit of building tools that observe the environment.
  2. It helps you connect programming to practical, everyday tasks.
  3. It can be expanded into a more complete dashboard later.
