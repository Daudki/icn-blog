Projects

Build and query a database for network-related information.

Network Inventory Database

Intermediate

Create tables for devices and store their hostnames, IP addresses, and locations.

inventory_db.py
import sqlite3
conn = sqlite3.connect('inventory.db')
conn.execute('CREATE TABLE IF NOT EXISTS devices (id INTEGER PRIMARY KEY, hostname TEXT, ip TEXT, location TEXT)')
conn.execute('INSERT INTO devices VALUES (1, "Core-SW1", "10.0.0.1", "Server Room")')
conn.commit()
print(conn.execute('SELECT * FROM devices').fetchall())

Device Maintenance Tracker

Beginner

Store maintenance events and retrieve them later with simple queries.

maintenance.py
import sqlite3
conn = sqlite3.connect('maintenance.db')
conn.execute('CREATE TABLE IF NOT EXISTS maintenance (id INTEGER PRIMARY KEY, device TEXT, note TEXT)')
conn.execute('INSERT INTO maintenance (device, note) VALUES (?, ?)', ('Router', 'Replaced cable'))
conn.commit()
print(conn.execute('SELECT * FROM maintenance').fetchall())
