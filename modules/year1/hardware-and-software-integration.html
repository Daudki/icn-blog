Projects

A practical introduction to talking with hardware from Python.

Serial Port Communicator

Intermediate

List available serial ports and open one to read data.

serial_comm.py
import serial

ports = serial.tools.list_ports.comports()
for port in ports:
    print(port.device, port.description)

Device Health Reporter

Beginner

Collect simple status updates from a connected device and display them.

health_report.py
status = {
    'power': 'on',
    'temperature': 'normal',
    'signal': 'stable'
}
for key, value in status.items():
    print(f'{key}: {value}')
