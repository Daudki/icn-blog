Overview

OOP helps you break down complex systems into smaller, reusable objects. This module focuses on how classes and inheritance make your code clearer and easier to extend.

Projects

Use classes to describe real-world networking concepts.

Network Device Class Hierarchy

Intermediate

Create base and child classes for routers, switches, and hosts.

devices.py
class NetworkDevice:
    def __init__(self, name, ip):
        self.name = name
        self.ip = ip

class Router(NetworkDevice):
    def __init__(self, name, ip, model):
        super().__init__(name, ip)
        self.model = model

print(Router('Core-R1', '192.168.1.1', 'Cisco 2900'))

Device Status Tracker

Beginner

Store online/offline status as a property that can be updated by each object.

status_tracker.py
class Device:
    def __init__(self, name):
        self.name = name
        self.online = False

    def ping(self):
        self.online = True

router = Device('Router')
router.ping()
print(router.online)
