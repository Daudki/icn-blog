Network Config Backup Tool

Intermediate

Simulate a simple configuration backup process for a router or switch.

config_backup.py
from datetime import datetime

config = "hostname Core-R1\ninterface Gi0/1\n ip address 192.168.1.1 255.255.255.0"
filename = f"backup_{datetime.now():%Y%m%d_%H%M%S}.txt"
with open(filename, 'w') as fh:
    fh.write(config)
print('Saved', filename)

VLAN Planner

Beginner

Build a small mapping for VLANs and their assigned departments.

vlan_planner.py
vlans = {
    10: 'Students',
    20: 'Staff',
    30: 'Servers'
}
for vlan_id, group in vlans.items():
    print(f'VLAN {vlan_id}: {group}')
